In this project we are taking as input pbf files from Geofabrik and we are extracting polygons. We only keep polygons which have a wikidata tag.


The GitHub associated with this project is: https://github.com/NoeFlandre/osm-polygon-wikidata-only
The data is stored remotely as a Hugging Face dataset at: https://huggingface.co/datasets/NoeFlandre/osm-polygon-wikidata-only
The data is stored locally on a hard drive at: '/Volumes/Seagate M3/projects/osm-polygon-wikidata-only'
